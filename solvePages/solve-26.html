Problem Statement

Write a C program to merge two already **sorted** integer arrays (Array A and Array B) into a single, new **sorted** array (Array C). This must be achieved by iterating through both source arrays only once, resulting in linear time complexity, $O(M+N)$, where $M$ and $N$ are the sizes of the arrays.

Input Format

Use two hardcoded sorted integer arrays (Array A and Array B).

Output Format

Print all three arrays (Array A, Array B, and the merged Array C), clearly labeled.

Examples:

Example 1
Input Arrays (Hardcoded)
Array A: {1, 3, 5, 7}
Array B: {2, 4, 6, 8}
Output
Array A: 1 3 5 7
Array B: 2 4 6 8
Merged Array C: 1 2 3 4 5 6 7 8
Constraints:
  • The merge must be done in $O(M+N)$ time complexity.
  • No general sorting algorithm (like Bubble or Quick Sort) is allowed on the final array C.
  • The result must be stored in a third array, C, of size $M+N$.
  • Arrays A and B must already be sorted in ascending order.

Hints:

1

Use three pointers: i for Array A, j for Array B, and k for Array C. Start all three at 0.

2

In a main loop, compare A[i] and B[j]. Copy the smaller element to C[k] and increment the respective source pointer (i or j) and the destination pointer (k).

3

After the main comparison loop ends, one array will have remaining elements. Use two separate loops to copy the leftover elements directly to the end of array C.

Explanation:

This method is the heart of the Merge Sort algorithm and guarantees $O(M+N)$ performance because each element is checked and placed exactly once. It relies entirely on the pre-sorted nature of the input arrays.

Initial pointers i, j, and k are set to 0. A loop runs as long as there are elements left in *both* arrays A and B.
The smaller element between A[i] and B[j] is copied to C[k]. The corresponding source pointer and k are incremented.
After the main loop finishes, two smaller loops check for any remaining elements in A or B (which happens if one array is longer) and copies them directly into C.

#include <stdio.h> // Function to merge two sorted arrays void mergeSortedArrays(int A[], int m, int B[], int n, int C[]) { int i = 0; // Index for A int j = 0; // Index for B int k = 0; // Index for C // Traverse both arrays until one is exhausted while (i < m && j < n) { if (A[i] <= B[j]) { C[k] = A[i]; i++; } else { C[k] = B[j]; j++; } k++; } // Copy remaining elements of A, if any while (i < m) { C[k] = A[i]; i++; k++; } // Copy remaining elements of B, if any while (j < n) { C[k] = B[j]; j++; k++; } } // Function to print the array void printArray(int arr[], int size, const char *name) { printf("%s: ", name); for (int i = 0; i < size; i++) { printf("%d ", arr[i]); } printf("\\n"); } int main() { int A[] = {1, 3, 5, 7}; int m = sizeof(A) / sizeof(A[0]); int B[] = {2, 4, 6, 8, 9}; int n = sizeof(B) / sizeof(B[0]); int total_size = m + n; int C[total_size]; // The merged array printArray(A, m, "Array A"); printArray(B, n, "Array B"); mergeSortedArrays(A, m, B, n, C); printArray(C, total_size, "Merged Array C"); return 0; }
