Problem Statement

Write a C program that defines a **struct** named `Student` with fields for `name` (a string), `id` (an integer), and `gpa` (a float). The program should then create a student variable, assign static values to its fields, and print them.

Input Format

No user input is required; the values for the student record are hardcoded (e.g., Alice Smith, ID: 101, GPA: 3.75).

Output Format

Print the student's record, including the name, ID, and GPA, clearly labeled and formatted (GPA to two decimal places).

Examples:

Example 1 (Student Record)
Assigned Values:
Name: Alice Smith, ID: 101, GPA: 3.75
Output
Student Record: Name: Alice Smith ID: 101 GPA: 3.75
Constraints:
  • The structure must be named `Student`.
  • The structure must contain at least a `char` array (for name), an `int` (for ID), and a `float` (for GPA).
  • Assignment to members must use the **dot operator (`.`)**.
  • The GPA field must be printed with 2 decimal places.

Hints:

1

Remember to use `struct Student s1;` to declare a variable of the structure type.

2

You cannot use the direct assignment operator (`=`) for the name field. You must use a string copying function like `snprintf` or `strcpy` (or manual copying).

Explanation:

The solution starts by defining the composite data type, `struct Student`, globally. In `main`, an instance of the struct, `s1`, is created. Values are assigned to the `id` and `gpa` members using direct assignment with the **dot operator**. For the string (`name`), a function like `snprintf` is necessary because C does not allow direct assignment to character arrays. Finally, the values are printed using the appropriate format specifiers, including `%.2f` for the GPA.

Define the structure globally or before `main`.
Declare a variable: `struct Student s1;`.
Assign integer and float values directly: `s1.id = 101;` and `s1.gpa = 3.75;`.
Use `snprintf` (or `strcpy` from ``) to safely assign the string value to the `name` array.
Print all members using the dot operator and correct `printf` formatting.

#include <stdio.h> // Define the Student structure struct Student { char name[50]; int id; float gpa; }; int main() { // Declare a structure variable struct Student s1; // --- Assign values to the structure members --- // Use snprintf to safely copy the string into the character array // snprintf is preferred over strcpy for buffer safety snprintf(s1.name, sizeof(s1.name), "Alice Smith"); // Assign integer and float members directly using the dot operator (.) s1.id = 101; s1.gpa = 3.75; // --- Print the structure members --- printf("Student Record:\\n"); printf("Name: %s\\n", s1.name); printf("ID: %d\\n", s1.id); printf("GPA: %.2f\\n", s1.gpa); // %.2f for two decimal places return 0; }
