9. Reverse Array (In-place)

Problem Statement

Write a C program that reverses the elements of a given array **in-place** (without using a second, auxiliary array). The program should initialize a sample array and print its elements before and after the reversal.

Input Format

The array is pre-defined in the code: int arr[] = {1, 2, 3, 4, 5};. No user input is required to run the main logic.

Output Format

Print the elements of the array before and after the in-place reversal, separated by spaces.

Examples:

Example 1 (Odd Length Array)
Pre-defined Array:
{1, 2, 3, 4, 5}
Output
Original array: 1 2 3 4 5 Reversed array: 5 4 3 2 1
Constraints:
  • The reversal must be **in-place** (O(1) auxiliary space complexity).
  • You must use a loop to perform the swap operation.
  • A temporary variable is required for swapping two elements.

Hints:

1

The core operation is swapping: element at index $i$ is swapped with the element at index $size - 1 - i$.

2

The loop only needs to run up to the halfway point of the array, or $i < size / 2$. Running the full length would reverse the array and then reverse it again back to its original state.

3

A helper function to print the array is useful for debugging and clean output.

Explanation:

The efficient, in-place array reversal is achieved by swapping elements from the two ends of the array until the indices meet in the middle. The key is to run the loop only up to $i < \text{size} / 2$ to ensure each pair is swapped only once. A temporary variable (`temp`) is essential to hold one value while the swap is performed.

Define a helper function, `print_array`, to display the array's contents.
Calculate the array size using the `sizeof` trick.
Use a `for` loop that iterates only up to the array midpoint (`size / 2`).
Inside the loop, swap `arr[i]` with `arr[size - 1 - i]` using a temporary variable `temp`.
Read This Code
#include <stdio.h> // Helper function to print array elements void print_array(int arr[], int size) { for (int i = 0; i < size; i++) { printf("%d ", arr[i]); } printf("\\n"); } int main() { int arr[] = {1, 2, 3, 4, 5}; // Odd-length array example int size = sizeof(arr) / sizeof(arr[0]); int temp; int i; printf("Original array: "); print_array(arr, size); // --- In-place Reversal --- // Loop only up to the midpoint (size / 2) for (i = 0; i < size / 2; i++) { // 1. Store element at start (arr[i]) temp = arr[i]; // 2. Overwrite start element with end element (arr[size - 1 - i]) arr[i] = arr[size - 1 - i]; // 3. Overwrite end element with stored start element (temp) arr[size - 1 - i] = temp; } // Print the reversed array printf("Reversed array: "); print_array(arr, size); return 0; }
