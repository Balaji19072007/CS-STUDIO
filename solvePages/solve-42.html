Problem Statement

Write a C program to implement the **Heap Sort** algorithm. Heap Sort is an in-place sorting algorithm that uses a Binary Heap structure. The implementation must include the helper function **heapify** (to maintain the heap property) and the main **heapSort** function. Assume a **Max Heap** is used for ascending order sorting.

Input Format

Use a hardcoded unsorted integer array.

Output Format

Print the array before and after sorting.

Examples:

Example 1
Input
(Array: {12, 11, 13, 5, 6, 7})
Output
Unsorted array: 12 11 13 5 6 7 Sorted array (Heap Sort): 5 6 7 11 12 13
Constraints:
  • The sort must be implemented using the **Max Heap** approach.
  • The solution must include a separate heapify function.
  • Sorting should be done **in-place** (without allocating a large auxiliary array).

Hints:

1

To convert an array into a Max Heap (Build Heap), you need to call heapify starting from the last non-leaf node, which is at index n/2 - 1, and iterate back to index 0.

2

The children of an element at index i are at 2*i + 1 (left) and 2*i + 2 (right).

3

The sorting phase involves swapping the root (largest element) with the last element of the unsorted subarray, then reducing the heap size and calling heapify on the new root.

Explanation:

Heap Sort consists of two main phases:

Build Max Heap: Convert the input array into a Max Heap. This is done by running heapify on all non-leaf nodes (from $\lfloor N/2 \rfloor - 1$ down to 0).
Heapify Down (Extraction): The largest element (at index 0) is swapped with the last element of the unsorted part, and the heap size is reduced. The new root is then heapified to restore the Max Heap property. This step is repeated until the heap size is 1.

The heapify function takes an array, its size, and the root index. It ensures the subtree rooted at i is a Max Heap.

#include <stdio.h> // Function to swap two elements void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; } // Function to heapify a subtree rooted with node i (N is the size of heap) void heapify(int arr[], int n, int i) { int largest = i; // Initialize largest as root int left = 2 * i + 1; // left child index int right = 2 * i + 2; // right child index // If left child is larger than root if (left < n && arr[left] > arr[largest]) { largest = left; } // If right child is larger than current largest if (right < n && arr[right] > arr[largest]) { largest = right; } // If largest is not root if (largest != i) { swap(&arr[i], &arr[largest]); // Recursively heapify the affected sub-tree heapify(arr, n, largest); } } // Main function to perform Heap Sort void heapSort(int arr[], int n) { // Phase 1: Build a Max Heap (rearrange array) // Start from the last non-leaf node for (int i = n / 2 - 1; i >= 0; i--) { heapify(arr, n, i); } // Phase 2: One by one extract an element from heap for (int i = n - 1; i > 0; i--) { // Move current root (largest element) to end swap(&arr[0], &arr[i]); // Call max heapify on the reduced heap heapify(arr, i, 0); } } // Function to print the array void printArray(int arr[], int size) { for (int i = 0; i < size; i++) { printf("%d ", arr[i]); } printf("\n"); } int main() { int arr[] = {12, 11, 13, 5, 6, 7}; int n = sizeof(arr) / sizeof(arr[0]); printf("Unsorted array: "); printArray(arr, n); heapSort(arr, n); printf("Sorted array (Heap Sort): "); printArray(arr, n); return 0; }
