Problem Statement

Write a C program that presents a simple menu to the user with three options: **1. Sum, 2. Subtract, 3. Exit**. The program should use a **`switch` statement** to perform the chosen operation on two pre-defined numbers.

Input Format

The program should prompt the user to enter their choice (1, 2, or 3).

Output Format

Based on the choice, print the result of the calculation (e.g., `Result: 30`) or an exit message (`Exiting the program. Goodbye!`). Invalid choices should also be handled.

Examples:

Example 1 (Choice: 1 - Sum)
Pre-defined Values: 10 and 20
Enter your choice (1-3): 1
Output
Simple Calculator Menu: 1. Sum 2. Subtract 3. Exit Enter your choice (1-3): 1 Result: 30
Example 2 (Choice: 2 - Subtract)
Pre-defined Values: 10 and 20
Enter your choice (1-3): 2
Output
Simple Calculator Menu: 1. Sum 2. Subtract 3. Exit Enter your choice (1-3): 2 Result: -10
Constraints:
  • Must use the `switch` statement based on the user's integer choice.
  • Pre-defined numbers must be $a=10$ and $b=20$.
  • The `switch` statement must include a `default` case to handle invalid input.
  • Each `case` block must terminate with a `break;` statement.

Hints:

1

Remember to print the entire menu *before* prompting for input.

2

The `case 3` operation should just print the exit message and does not need arithmetic calculation.

Explanation:

The core of this problem is the **`switch` statement**, which evaluates the `choice` variable and jumps directly to the matching `case` block. Using `break` after every `case` is vital to prevent **fall-through**, where execution continues into the next case unintentionally. The `default` case ensures that invalid inputs are handled gracefully.

Define the initial variables (`num1 = 10`, `num2 = 20`) and `choice`.
Print the menu options and read the `choice` using `scanf()`.
Implement the `switch (choice)` block with `case 1`, `case 2`, and `case 3`.
Ensure each valid `case` calculates and prints the result, followed by a `break;`.
Implement the `default` case for inputs outside the 1-3 range.

#include <stdio.h> int main() { int choice; int num1 = 10, num2 = 20; int result; printf("Simple Calculator Menu:\\n"); printf("1. Sum\\n"); printf("2. Subtract\\n"); printf("3. Exit\\n"); printf("Enter your choice (1-3): "); if (scanf("%d", &choice) != 1) { printf("Invalid input type. Exiting.\\n"); return 1; } // Use a switch statement to handle the choice switch (choice) { case 1: result = num1 + num2; printf("Result: %d\\n", result); break; // Essential to prevent fall-through case 2: // Subtract num2 from num1 result = num1 - num2; printf("Result: %d\\n", result); break; // Essential to prevent fall-through case 3: printf("Exiting the program. Goodbye!\\n"); break; default: printf("Invalid choice. Please enter 1, 2, or 3.\\n"); // No break needed for default as it's the last block } return 0; }
