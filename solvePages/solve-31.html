31. Reverse Words in a String

Problem Statement

Write a C program to reverse the order of **words** in a given string. The individual characters within each word should remain in their original order. The words are separated by a single space.

Input Format

Use a hardcoded string literal or character array containing words separated by single spaces.

Output Format

Print the original string and the string with the words reversed.

Examples:

Example 1
Input String (Hardcoded)
"Gemini is a helpful assistant"
Output
Original String: Gemini is a helpful assistant
Reversed Words: assistant helpful a is Gemini
Constraints:
  • Words are separated by a single space character.
  • The original string can contain up to 100 characters.
  • The solution should be designed to handle multiple words (at least 2).
  • Standard C string library functions (like strtok, strcpy) are generally permitted for this level of challenge.

Hints:

1

The strtok() function is ideal for splitting a string into tokens (words) based on a delimiter (the space character).

2

Since strtok modifies the original string, make a copy of the input string first if you need the original for display later.

3

Store the pointers returned by strtok() in an array of character pointers (char *words[]). Then, loop through this array in reverse order to print the words.

Explanation:

The solution uses the standard C library function strtok to easily tokenize the string based on spaces, and an array of pointers to keep track of the start of each word without creating new storage for the words themselves.

First, a copy of the string is made using strcpy because strtok modifies the string in place by inserting null terminators.
strtok is called repeatedly inside a loop. Each call finds the next word, and the starting address (pointer) of that word is saved into the words array.
Finally, a for loop iterates backward through the words array (from word_count - 1 down to 0), printing the strings in reverse order to achieve the desired result.

#include <stdio.h> #include <string.h> #include <stdlib.h> #define MAX_WORDS 20 // Function to reverse the order of words void reverseWords(char *str) { // Array to store pointers to each word char *words[MAX_WORDS]; int word_count = 0; // Use strtok to tokenize the string (words separated by space) char *token = strtok(str, " "); while (token != NULL && word_count < MAX_WORDS) { words[word_count++] = token; // Store the pointer to the word token = strtok(NULL, " "); // Get the next token } // Print the words in reverse order printf("Reversed Words: "); for (int i = word_count - 1; i >= 0; i--) { printf("%s", words[i]); if (i > 0) { printf(" "); // Print space between words } } printf("\\n"); } int main() { // Note: Must use a char array to allow modification by strtok. char str[] = "Gemini is a helpful assistant"; printf("Original String: %s\\n", str); // We need a temporary copy because strtok modifies str char temp_str[100]; strcpy(temp_str, str); // The function modifies temp_str and prints the result reverseWords(temp_str); return 0; }
