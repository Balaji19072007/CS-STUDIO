Problem Statement

Write a C program to calculate the **transpose** of a 2D matrix. Given a matrix $A$ of size $M \times N$, its transpose $A^T$ is an $N \times M$ matrix where the element at row $i$ and column $j$ in $A^T$ is the element at row $j$ and column $i$ in $A$. ($A^T[i][j] = A[j][i]$).

Input Format

Use a hardcoded $3 \times 2$ matrix for the original array $A$.

Output Format

Print the original matrix and its transposed matrix, with elements correctly spaced and rows separated by newlines.

Examples:

Example 1
Original Matrix A (3x2)
1 2
3 4
5 6
Transposed Matrix Aᵀ (2x3)
1 3 5
2 4 6
Constraints:
  • The program must use two separate 2D arrays (matrices) to store the original and the transpose.
  • The original matrix should be declared with dimensions $M \times N$ (e.g., $3 \times 2$).
  • The transposed matrix must be declared with dimensions $N \times M$ (e.g., $2 \times 3$).
  • The solution should use nested loops to perform the element copy.

Hints:

1

Use preprocessor directives (#define) for ROWS and COLS to make the code cleaner and easier to modify.

2

The core transformation logic inside the loops is: B[j][i] = A[i][j], where $A$ is $M \times N$ and $B$ is $N \times M$.

3

When printing the result matrix, remember to use the swapped dimensions (outer loop iterates up to $N$, inner loop up to $M$).

Explanation:

Transposing a matrix simply means converting rows into columns and columns into rows. This requires creating a new matrix with swapped dimensions and then copying elements in the reversed order.

Define ROWS (M=3) and COLS (N=2) and declare two arrays: $A[M][N]$ and $B[N][M]$.
Use nested loops: the outer loop runs $M$ times (rows of $A$), and the inner loop runs $N$ times (columns of $A$).
Inside the loops, the transformation $B[j][i] = A[i][j]$ is performed, effectively copying the element from the $(i, j)$ position in $A$ to the $(j, i)$ position in $B$.
Read This Code
#include <stdio.h> #define ROWS 3 #define COLS 2 // Function to print a matrix (M x N) void printMatrix(int matrix[ROWS][COLS], int r, int c) { for (int i = 0; i < r; i++) { for (int j = 0; j < c; j++) { printf("%d ", matrix[i][j]); } printf("\\n"); } } int main() { // Original Matrix (3x2) int A[ROWS][COLS] = { {1, 2}, {3, 4}, {5, 6} }; // Transposed Matrix will be 2x3 (COLS x ROWS) int B[COLS][ROWS]; printf("Original Matrix (%dx%d):\\n", ROWS, COLS); // Note: We cannot pass A directly to a function expecting B, so we print inline for this example. for (int i = 0; i < ROWS; i++) { for (int j = 0; j < COLS; j++) { printf("%d ", A[i][j]); } printf("\\n"); } // --- Transpose Calculation --- for (int i = 0; i < ROWS; i++) { // i is the row index of A for (int j = 0; j < COLS; j++) { // j is the col index of A // A[i][j] becomes B[j][i] B[j][i] = A[i][j]; } } // --- End of Transpose Calculation --- printf("\\nTransposed Matrix (%dx%d):\\n", COLS, ROWS); // Note: Printing B uses its actual dimensions: COLS x ROWS for (int i = 0; i < COLS; i++) { for (int j = 0; j < ROWS; j++) { printf("%d ", B[i][j]); } printf("\\n"); } return 0; }
