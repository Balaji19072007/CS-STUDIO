Problem Statement

Write a C program that prompts the user to enter two integers. The program should calculate and print the sum and the product of these two numbers.

Input Format

The program should prompt the user to enter the first integer, and then the second integer.

Output Format

The program must print the two input numbers, their calculated sum, and their calculated product, clearly labeled.

Examples:

Example 1
Input
Enter the first integer: 15 Enter the second integer: 7
Output
First Number: 15 Second Number: 7 Sum: 22 Product: 105
Constraints:
  • Both inputs must be standard C integers (int)
  • Output must clearly label the input values, sum, and product
  • Basic error handling for invalid input is recommended

Hints:

1

Use the %d format specifier in both scanf() and printf().

2

Remember to use the address-of operator (&) when reading inputs with scanf().

3

Store the calculated sum and product in their own variables before printing.

Complete Solution

Ready-to-use code in C

Explanation:

This is the simplest foundation of C programming. It involves three key steps:

Declare four integer variables: two for input (num1, num2) and two for results (sum, product).
Use printf() and scanf() with the %d format specifier to read the two integer values. The solution includes basic error handling for scanf().
Perform the simple arithmetic operations: sum = num1 + num2; and product = num1 * num2;.
Use four separate printf() statements to display the original inputs and the calculated results in the specified format.
Read This Code
#include <stdio.h> int main() { // 1. Declare variables to store the inputs and results int num1, num2; int sum, product; // 2. Prompt and read the first integer printf("Enter the first integer: "); if (scanf("%d", &num1) != 1) { printf("Invalid input. Exiting.\n"); return 1; // Basic error handling } // 3. Prompt and read the second integer printf("Enter the second integer: "); if (scanf("%d", &num2) != 1) { printf("Invalid input. Exiting.\n"); return 1; // Basic error handling } // --- Start of Calculations --- // Calculate the sum sum = num1 + num2; // Calculate the product product = num1 * num2; // --- End of Calculations --- // 4. Print the results printf("\nFirst Number: %d\n", num1); printf("Second Number: %d\n", num2); printf("Sum: %d\n", sum); printf("Product: %d\n", product); return 0; }
