7. Array Sum/Average

C Easy

Problem Statement

Write a C program that initializes an array of **5 integers**, reads the elements from the user, calculates the **sum** of all elements, and determines the **average**.

Input Format

The program should prompt the user to enter 5 integers one by one (or separated by newlines/spaces).

Output Format

The program must print both the total sum and the calculated average (formatted to two decimal places, e.g., XX.XX).

Examples:

Example 1 (Standard Integers)
Input
Enter 5 integers: 10 20 30 40 50
Output
Sum of elements: 150 Average of elements: 30.00
Example 2 (Mixed Integers)
Input
Enter 5 integers: 1 2 3 4 5
Output
Sum of elements: 15 Average of elements: 3.00
Constraints:
  • The array size must be 5.
  • You must use a `for` loop to read input and another `for` loop for summation.
  • Type casting to a floating-point number is mandatory for the average calculation to be accurate.

Hints:

1

Declare the sum as an `int` and the average as a `float` or `double`.

2

The formula for average is sum / count. To avoid integer division truncation, ensure the `sum` variable is cast to float/double *before* the division: (float)sum / count.

3

Use the %.2f format specifier in printf() to ensure the average is printed with exactly two decimal places.

Explanation:

We use a single array and iterate through it with a for loop to both read the elements from the user and, simultaneously, calculate the running sum. To calculate the average accurately, we perform explicit **type casting** to `float` when dividing the sum by the array size, ensuring floating-point division is used instead of integer division.

Declare an integer array of size 5, an integer `sum` initialized to 0, and a `float` or `double` variable for `average`.
Use a for loop to read all 5 elements using `scanf()` and accumulate the total sum.
Calculate the average using the formula: `average = (float)sum / array_size;`
Print the results, ensuring the average uses the `%.2f` format specifier.
